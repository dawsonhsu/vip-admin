# VIP 簽到日誌 — 後台功能說明

## 頁面用途
查詢所有會員的每日簽到行為與獎勵觸發記錄，供運營與客服人員追蹤簽到狀態、排查異常。

---

## 統計卡片（頂部）

| 指標 | 說明 |
|------|------|
| 簽到總筆數 | 當前篩選條件下的簽到記錄總數 |
| 補簽筆數 | 簽到類型為「補簽」的記錄數 |
| 獎勵觸發筆數 | 觸發了獎勵節點（第 6/7/16/17/26/27 天）的記錄數 |
| 當日簽到人數 | 今日完成簽到的會員人數 |

---

## 篩選條件

| 欄位 | 類型 | 說明 |
|------|------|------|
| 會員帳號 | 文字輸入 | 模糊搜尋，不分大小寫 |
| 會員 UID | 文字輸入 | 精確包含搜尋 |
| VIP 等級 | 下拉選單 | V0 ~ V30 |
| 等級區間 | 下拉選單 | Bronze / Silver / Gold / Platinum / Diamond |
| 簽到類型 | 下拉選單 | 一般 / 補簽 |
| 是否觸發獎勵 | 下拉選單 | 是 / 否 |
| 簽到時間 | 日期範圍 | 起迄日期 |

---

## 表格欄位

| 欄位 | 說明 |
|------|------|
| 序號 | 記錄 ID |
| 會員帳號 | 可點擊跳轉會員詳情 |
| 會員 UID | 會員唯一識別碼 |
| VIP 等級 | 簽到當下的 VIP 等級（V0-V30） |
| 等級區間 | 對應頭銜：Bronze / Silver / Gold / Platinum / Diamond |
| 簽到日 | 當前連續簽到的第幾天（支援排序） |
| 簽到前連續天數 | 本次簽到前已累計的連續天數 |
| 簽到後連續天數 | 本次簽到後的連續天數（= 簽到前 + 1） |
| 簽到類型 | 一般簽到 / 補簽 |
| 補簽存款金額 | 補簽時需滿足的存款金額，一般簽到顯示「-」 |
| 觸發獎勵節點 | 若本次簽到觸發了獎勵，顯示 Day 6/7/16/17/26/27，否則「-」 |
| 當前週期 | 第幾輪簽到週期（每 27 天為一輪） |
| 簽到時間 | 簽到操作的時間戳（YYYY-MM-DD HH:mm:ss） |

---

## 簽到規則摘要

| 規則 | 說明 |
|------|------|
| 簽到週期 | 27 天一輪，第 28 天重新從第 1 天開始 |
| 獎勵節點 | 第 6、7、16、17、26、27 天可領取彩金（第 7/17/27 天額外含 Free Spins） |
| 第 1-5 / 8-15 / 18-25 天 | 無簽到獎勵 |
| 補簽 | 每月限 4 次（Diamond 限 5 次），需滿足對應存款金額 |
| 補簽前置檢查 | 簽到前必須先補完所有漏簽天數，否則連續天數從第 1 天重新計算 |
| 自動重置 | 若未簽到天數超過可補簽上限，簽到自動重置 |
| VIP 依據 | 以簽到當天的 VIP 頭銜為準 |
| 流水要求 | 彩金與 Free Spins 各有獨立流水倍數要求 |
| 需手動簽到 | 會員必須手動點擊簽到按鈕 |

---

## 操作功能

| 功能 | 說明 |
|------|------|
| 查詢 | 根據篩選條件搜尋記錄 |
| 重置 | 清空所有篩選條件 |
| 導出 Excel | 匯出當前篩選結果為 Excel 檔案 |
| 分頁 | 每頁 20 筆，可調整每頁筆數 |
